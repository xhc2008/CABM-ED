## 核心系统解析

### 🗣️ 对话引擎
- 接入LLM模型，对话流畅得就像在跟真人聊天
- 情感系统：她的情绪状态由AI实时控制——别惹她生气，后果自负
- 好感度系统：-65536~65535的范围，足够细腻到记录你的每一个操作
- 多语言语音：中英日随意切换，语音合成效果堪比专业级
- 主动对话：长时间不操作？小心她主动找你「算账」

### 🧠 记忆矩阵
- 突破了传统上下文窗口的限制，实现了真正的长期记忆
- 详细技术说明见：[关于记忆力的说明](docs/关于记忆力的说明.md)
- *（小声）比某些人类的记忆力强多了*

### 🏠 多场景渲染
- 右侧快速切换场景——UI设计简洁得像一段优雅的代码
- 每个场景都有专属动作，细节控的福音

### 📔 日志系统
- 自动记录所有对话，支持语音回放
- 离线日志：你不在的时候，她也在默默记录生活
- *就像系统的日志文件，但有趣多了*

### 👗 皮肤系统
- 入口：浴室镜子
- 多套服装，场景自适应
- *偷偷告诉你，有你想看的……*

### 🎮 游戏模块
- 入口：客厅点击雪狐
- 当前版本：五子棋、中国象棋
- *赢了她可能会提高好感度，输了……你自己体会*

### 🎵 音频引擎
- 入口：书房电脑
- 支持自定义BGM，不过只支持mp3,wav和ogg格式

### 🌱 休闲玩法集
- **菜园**（天台地面）：种菜收菜，养老玩法
- **做饭**（厨房灶台）：自由组合食材
- **超市**（超市门）：每日刷新物品

### 🗺️ 探索模式（Preview）
- 入口：玄关/天台/超市打开地图
- 功能特性：
  - 随机物资箱——开箱的快乐，懂的都懂
  - 拾取、战斗，动作流畅
  - 雪狐AI：跟随、攻击、~~捡垃圾~~
  - 敌人AI：会主动攻击，击败掉落
- 地图每日刷新，保证新鲜感
- 不过注意，如果不小心死了会爆装备的
