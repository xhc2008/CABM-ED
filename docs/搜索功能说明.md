# 日记搜索功能

## 功能说明

为日记查看器添加了全局搜索功能，可以根据关键词搜索所有日期的日记内容。

## 使用方法

1. 打开日记查看器
2. 在顶部搜索框中输入关键词
3. 点击"🔍 搜索"按钮或按回车键
4. 查看搜索结果，关键词会以黄色高亮显示
5. 点击"✖ 清除"按钮返回正常浏览模式

## 搜索范围

- **offline类型**：搜索事件内容
- **games类型**：搜索游戏事件内容
- **chat类型**：搜索对话总结和详细对话内容

## 搜索特性

- 不区分大小写
- 支持中英文关键词
- 搜索结果按日期降序排列
- 关键词在结果中黄色高亮显示
- 显示搜索结果数量
- chat类型记录可点击查看详细对话

## 界面变化

- 在标题和关闭按钮下方添加了搜索栏
- 搜索栏包含：
  - 搜索输入框（支持回车提交）
  - 搜索按钮
  - 清除按钮
- 搜索时隐藏日期选择器
- 搜索结果卡片顶部显示日期标签（📅）

## 技术实现

- 使用 `RichTextLabel` 支持BBCode高亮显示
- 遍历所有日期的日记文件进行搜索
- 搜索结果保留原有的记录类型和交互功能
